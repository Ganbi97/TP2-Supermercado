Para encriptar la contraseña vamos a necesitar:
Char password[10]; // esta variable contendrá la contraseña a encriptar
Int matriz[2][5]; // en esta matriz pondremos los números ASCII correspondientes a la contraseña
Int mDim; // en esta variable se guardara la dimensión de la matriz por si la contraseña no llega a 10 char.

Entonces llamamos a la función de tipo int:

mDim=fPass2Matrix(password, matriz); // esta función pasa la contraseña a la matriz ASCII y luego invocamos a la función de tipo void:

fMatrizEncrypting(matriz,usuario.pass,mDim); // donde matriz es la matriz ASCII – usuario.pass es la matriz de la estructura donde guardaremos la contraseña encriptada y mDim es la dimensión de la matriz (que fue devuelta por fPass2Matrix)


Para desencriptar:

fDecrypting(pass, matrizPass); // a esta función de tipo void le pasamos un char pass[10] donde se desencriptará la contraseña y la matriz de 2x5 que contiene el password encriptado. La contraseña se desenriptara y será guardada en pass.
